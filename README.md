## ⚙️ Run Locally

```bash
# Clone the repo
git clone https://github.com/Akshaya-Vasireddi/iris-ML-classifier.git
cd iris-ML-classifier

# Install dependencies
pip install -r requirements.txt

# Launch the app
streamlit run app.py
```
